## Phase 3: 전체 시스템 확장

> **목표**: 나머지 11개 시스템(이중화 포함)에 에이전트를 배포하고 Prometheus scrape 설정 전체 확장

### T3.1 — 전체 시스템 에이전트 배포 (자동 스크립트 활용)

**배포 실행 예시 (13개 시스템 전체)**:

```bash
# ---- 모니터링 서버에서 각 대상 서버로 에이전트 패키지 전송 ----
for server in cx-was01 cx-was02 cx-web01 cx-web02 pos-svr01 pos-svr02 erp-svr01; do
    scp -r /opt/aoms-offline/agents/linux/ user@${server}:/tmp/aoms-agents/
    scp /opt/aoms/agents/linux/install-agents.sh user@${server}:/tmp/aoms-agents/
done

# ---- 각 서버에서 설치 실행 (ssh 일괄 실행) ----
# 고객경험 WAS1
ssh user@cx-was01 "sudo /tmp/aoms-agents/install-agents.sh \
    --system-name customer-experience --instance-role was1 --host cx-was01 \
    --monitoring-server 192.168.10.5 --install-dir /opt/aoms-agents \
    --log-path '/app/logs/*.log' --type all --jmx-port 9404"

# 고객경험 WAS2
ssh user@cx-was02 "sudo /tmp/aoms-agents/install-agents.sh \
    --system-name customer-experience --instance-role was2 --host cx-was02 \
    --monitoring-server 192.168.10.5 --install-dir /opt/aoms-agents \
    --log-path '/app/logs/*.log' --type all --jmx-port 9404"

# 고객경험 WEB1 (Nginx)
ssh user@cx-web01 "sudo /tmp/aoms-agents/install-agents.sh \
    --system-name customer-experience --instance-role web1 --host cx-web01 \
    --monitoring-server 192.168.10.5 --install-dir /opt/aoms-agents \
    --log-path '/var/log/nginx/*.log' --type node,promtail"

# POS 시스템 (Windows)
# → PowerShell 원격 실행 또는 수동 실행
```
