
**PromQL 예시 (이중화 대응)**:

```promql
# 선택한 시스템의 서버별 CPU
(1 - avg by(instance_role, host) (
    rate(node_cpu_seconds_total{mode="idle", system_name="$system_name"}[5m])
)) * 100
```

### T5.4.1 — Qdrant 이상 패턴 클러스터링 (주 1회)

```python
# 주 1회 k-means 클러스터링으로 반복 이상 패턴 자동 분류
# n8n Schedule 워크플로우로 실행 (매주 일요일 02:00)
#
# 1. Qdrant에서 최근 7일 벡터 샘플 조회 (최대 10,000건)
# 2. MiniBatchKMeans (k=20) 클러스터링
# 3. 클러스터 대표 벡터 + 레이블을 Qdrant 'pattern_clusters' 컬렉션에 저장
# 4. 신규 이상 발생 시 가장 가까운 클러스터에 매핑 → 패턴명 자동 부여
```

| 항목 | 내용 |
|------|------|
| 실행 주기 | 매주 일요일 02:00 (n8n Schedule) |
| 입력 | Qdrant `log_incidents` 최근 7일 벡터 |
| 출력 | `pattern_clusters` 컬렉션 (클러스터 대표 벡터 20개) |
| 효과 | 반복 장애 패턴 자동 명명 → LLM 프롬프트에 패턴명 포함 |
